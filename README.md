## Installation:
Node Package Manager (NPM)
```bash
npm i xtra-logger
```

Yet Another Resource Navigator (YARN)

```bash
yarn add xtra-logger
```

<br>

## Importing:

For Es5 Js

```js
const { logger, format } = require("xtra-logger");
```

For Es6 Js / Typescript

```js
import { logger, format } from "xtra-logger";
```
